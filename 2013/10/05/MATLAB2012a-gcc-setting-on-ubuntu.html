Matlab R2012a GCC Setting in Ubuntu

October 5th, 2013

This file describes how to configure Matlab to compile c/c++ code in Ubuntu.

  1. Let mcc use gcc-4.4

The version of gcc that Matlab R2012a works with is “4.4” which is lower than the default version in Ubuntu.

1.1 Install gcc-4.4 and gfortran-4.4

 
   sudo apt-get install gcc-4.4 gfortran-4.4

1.2 Set mcc to use the lower version of gcc and gfortran: Edit file ~/.matlab/R2012a/mexopts.sh (need root), replace gcc with gcc-4.4, g++ with g++-4.4 and gfortran with gfortran-4.4.

  1. Let MATLAB use the dynamic library of OS

    After step 1, there may be errors still in linking step. The simplest way of solving this is to rename (or remove) the dynamic libraries used in folder ~/MATLAB/R2012a/sys/os/glnxa64/, for example,

   cd ~/MATLAB/R2012a/sys/os/glnxa64/
   mv libXm.so.3 ./libXm.so.3.backup

Do this for all the .so files that meet error.
